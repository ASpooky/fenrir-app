## 利用方法と注意点

### .env について

    KEY=OOOOOOOOOOOOO(グルメサーチに必要なAPIキー)

/fenrir_back/.env に上記のファイルを配置してください。

### フロントエンド

fenrir_front にて以下を実行してください.

    npm install
    npm run dev

### バックエンド

fenrir_back にて以下を実行してください.

    go run main.go
